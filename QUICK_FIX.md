# Quick Fix for GitHub Actions Deployment

## Most Common Issue: Permissions

**Fix Steps:**

1. Go to: https://github.com/suduli/UDS-SIMULATION/settings/pages
   - Set **Source** to: **GitHub Actions**

2. Go to: https://github.com/suduli/UDS-SIMULATION/settings/actions
   - Under **Workflow permissions**, select: **Read and write permissions**
   - Click **Save**

3. Re-run the workflow:
   - Go to: https://github.com/suduli/UDS-SIMULATION/actions
   - Click the failed workflow
   - Click **Re-run all jobs**

## Expected Deployment URL
https://suduli.github.io/UDS-SIMULATION/
